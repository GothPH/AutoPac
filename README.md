# AutoPac
Simple script to search for your query and install all found results automatically #Pacman




# Installation
```bash
git clone https://github.com/SlavPH/AutoPac && cd AutoPac
```

# Usage
```bash
sudo python3 autopac.py [arg1] [arg2] ...
```
